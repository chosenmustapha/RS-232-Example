# RS-232-Example
Turning two leds on or off via the computer.

This program demonstrate how serial communication work.
It relies on the computer for its input and the output is comming from the Arduino Pins.
The Leds are on based on what is defined in the code.
When you enter 1: LED1 is on , LED2 is off. when you enter 0: LED2 is on, LED1 is off. Any other inputs is invalid.
